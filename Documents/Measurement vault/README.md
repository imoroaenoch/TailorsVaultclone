# Tailors Vault

A simple, mobile-first web application for tailors and fashion designers to quickly record and retrieve client measurements.

## Features

- **Quick Measurement Entry**: Record client measurements in under 30 seconds
- **Fast Search**: Find saved measurements by client name or phone number
- **Mobile-First Design**: Optimized for use on mobile devices while measuring clients
- **No Authentication**: Simple single-user app, no login required
- **Local Storage**: All data stored locally in your browser

## Setup

1. Simply open `index.html` in a web browser
2. No installation or build process required
3. Works offline after initial load

## Usage

### Recording a New Measurement

1. Click **"New Measurement"** on the home screen
2. Enter the client's name (required)
3. Optionally enter phone number, garment type, and notes
4. Enter measurement values (Shoulder, Chest, Waist, Sleeve, Length, Neck)
5. Click **"Save"** to store the measurement

### Searching for Measurements

1. Click **"Search Measurements"** on the home screen
2. Type the client's name or phone number in the search box
3. Click on a client from the results to view all their measurements
4. Measurements are grouped by date and show garment type and notes

## Data Storage

All data is stored locally in your browser using localStorage. This means:
- Data persists between sessions
- No internet connection required
- Data is private to your browser

**Note**: If you clear your browser data, all measurements will be lost. Consider exporting important data regularly.

## Browser Compatibility

Works on all modern browsers:
- Chrome/Edge (recommended)
- Firefox
- Safari
- Mobile browsers (iOS Safari, Chrome Mobile)
